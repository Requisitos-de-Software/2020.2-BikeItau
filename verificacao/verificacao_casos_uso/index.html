    Verificação de Casos de Uso

    Introdução

    Este documento apresenta a verificação do artefato de casos de uso, visando elucidar possíveis falhas presentes na confecção inicial de tal documento, além de atestar sua qualidade geral.

    Metodologia

    Foi utilizado um sistema de checklists para essa análise, o qual utiliza questões como critérios de aceitação para a verificação. A seguir são apresentadas tais questões:

    1. O título do caso de uso é claro e descritivo?
    2. Os atores estão claramente definidos
    3. As pré-condições foram estabelecidas?
    4. As pós-condições foram estabelecidas?
    5. Os fluxos básico e alternativo foram claramente descritos?
